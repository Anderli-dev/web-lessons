Розмітка коду

<code> - призначеня для відображення коду програм

Приклади використання:

<code> if (!createShader(fragmentShader, GL_FRAGMENT_SHADER, fragmentShaderID)) {
cerr << "fragment SHADER Complite-time error:\n" << endl; glDeleteShader(vertexShaderID); return; }
</code>

Результат:

if (!createShader(fragmentShader, GL_FRAGMENT_SHADER, fragmentShaderID)) {
cerr << "fragment SHADER Complite-time error:\n" << endl; glDeleteShader(vertexShaderID); return; }

<kbd> - призначеня для клавіш котрі треба натиснути користувачу

Приклади використання:

Цитати та визначення

<dfn> - використовується для означення визначень

Приклади використання:

<dfn>HTML> - мова розмітки гіпертексту</dfn>

Результат:

HTML- мова розмітки гіпертексту

<abbr> - якщо навести на абривіатуру то вона розшифрується

Приклади використання:

<p><abbr title="Hypertext Markup Language">HTML</abbr> - мова розмітки гіпертексту</p>

Результат:

HTML

<dfn> - використовується для означення цитати

Приклади використання:

<p>А Дмитро Джалябов сказав <q>Не знаю</q></p>

Результат:

А Дмитро Джалябов сказав Не знаю

<dfn> - використовується для означення цитати з посиланням на щось

Приклади використання:

<p><cite>А Дмитро Джалябов сказав <cite><q>Не знаю</q></p>

Результат:

А Дмитро Джалябов сказав Не знаю

<dfn> - використовується для означення цитати з посиланням на щось

Приклади використання:

<blockquote>Це зроблено не на 3 і не на 4, а на о%&@ь!</blockquote>

Результат:

Це зроблено не на 3 і не на 4, а на о%&@ь!

<br> - використовується для перенесення на інший рядок

У сні?
